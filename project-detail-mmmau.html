Mamamoo Among Us

With the explosion of people suddenly discovering Among Us for the first time, I decided to jump on the bandwagon and design an Among Us map using Mamamoo, a K-pop group, albums and sets. Instead of playing as Among Us, the goal was to create "tasks" that help users discover more about Mamamoo and learn about their music, videos, and themes. The technical goal I had in this project is I wanted to make every "task" as modular as posible. Tasks would listen to events fired and according to the event name, the respective task would appear. This would make it easy to add task in the future and allow for others to help make tasks should more people join my project.

Created in Unity. Available via Unity WebGL (Desktop mainly).
