Ddun Ddun Ddun

I took an interest in creating web games and wanting to learn some basic Javascript. At the same time, this song, Ddun ddun ddun, came out and I combined those two desires into this mini game where the map is created using the lyrics of the song and the player will need to try to collect as many coins as they can without getting caught by the scrolling and before the end of the song.

Created with Phaser. Available on Desktop browsers only.
