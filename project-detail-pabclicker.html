Level up Clicker

I have always been a fan of idle games, so I wanted to make one for myself. Though the idea seemed really simple, the math behind large numbers blew out my brain. So instead of doing something like mining cookies, I went for a more Pokemon battle mechanic but as an idle clicker. For this project, I created a queue for all damaged received so that even if the user outclicks the animation, the UI will still show all damage done. The future main mechanic of this game will be using your resources wisely in leveling up the correct cards to defeat the enemy in the allotted time.

Created in Unity. Available via Unity WebGL (Desktop mainly).
